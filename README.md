# Product Management System
## Problem Statement
- Manual product management is inefficient and error-prone.
- There is a need for a centralized system to manage products digitally
with CRUD operations and simple UI.
## Features
- Add new products
- View product list
- Update product details
- Delete products
- REST API based backend
- Simple HTML + JS frontend
## Technology Stack
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript
- Maven
## Project Structure
- Controller - handles HTTP requests
- Service - business logic
- Repository - database access
- Model - entity classes
- DTO - data transfer object
## Frontend Resources
- static folder
- index.html
- styles.css
- app.js
## Configuration
- application.properties
   - Database and server configuration
## API Flow Diagram
- Browser UI
        ↓
- Controller (ProductController)
        ↓
- Service (ProductService)
        ↓
- Repository (ProductRepository)
        ↓
- MySQL Database
